# **App Name**: Krishi-Sahayak

## Core Features:

- Multilingual Support: Allow users to switch between Hindi and English, storing the preference for future sessions using local storage. All text elements are updated instantly.
- AI Crop Recommendation Tool: Suggest top 3 crops based on land details (soil type, area, previous crop), using an AI tool to analyze available data and parameters.
- Voice Input: Enable voice input in Hindi and English for describing crop symptoms and other queries.
- Disease Detection: Allow farmers to upload crop images and describe symptoms. The system analyzes this to suggest potential causes and solutions.
- Voice Output of Solutions: Provide voice output of the cause and solution to a detected disease in the user's selected language (Hindi/English).
- Mandi Price Charts: Display live Mandi prices and demand trends for selected crops via interactive charts.

## Style Guidelines:

- Primary color: Earthy Green (#8FBC8F) to reflect agriculture and nature.
- Background color: Light Beige (#F5F5DC) to provide a neutral and calming backdrop.
- Accent color: Golden Yellow (#FFD700) to highlight key information and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern and accessible feel.
- Use clear, representative icons for crops, weather conditions, and market trends.
- Maintain a clean, card-based layout for easy readability and navigation across different sections.
